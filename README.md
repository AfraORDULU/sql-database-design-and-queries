# SQL Database Design and Queries

Bu projede temel bir çalışan veritabanı tasarlanmış ve SQL sorguları
kullanılarak veri analizi yapılmıştır.

## İçerik
- Tablo oluşturma (CREATE TABLE)
- Veri ekleme (INSERT)
- GROUP BY ile özet sorgular
- Filtreleme (WHERE)

## Kullanılan Teknolojiler
- SQL
- MySQL / SQLite uyumlu
